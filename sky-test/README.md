#  Cine Sky

## Objetivo

Criar um app de duas telas, uma inicial com uma lista de filmes trazidos pela API **https://sky-exercise.herokuapp.com/api/Movies** que retorna a seguinte estrutura de dados:

~~~javascript
{
    title: "",
    overview: "",
    duration: "",
    release_year: "",
    cover_url: "",
    backdrops_url: [
        "",
    ],
    id: ""
}
~~~
